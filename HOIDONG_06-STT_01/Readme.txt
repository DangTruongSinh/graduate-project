Thư mục "Noi dung" chứa các file toàn văn báo cáo ở dạng *.doc và *.pdf để in ra
quyển đồ án.
Thư mục "Ma nguon" chứa các file sau:
- "Fingerprint.ino" dùng để nạp cho adruino ở node bảo mật.
- "Gateway.py" dùng để nạp cho raspberry pi dùng để làm gateway của hệ thống.
- "SecurityNode.py" dùng để nạp cho rasberry pi dùng làm node bảo mật của hệ thống.
- "SensorNode1.ino" và "SensorNode2.ino" là 2 file dùng để nạp cho 2 adruino ở node cảm biến của hệ thống.
- "SecurityNode.pdsprj" sơ đồ mạch nguyên lý và mạch in của node bảo mật.
- "SensorNode.pdsprj" sơ đồ mạch nguyên lý  và mạch in của node cảm biến.
