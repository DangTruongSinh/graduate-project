Trong file thư mục HOIDONG_06-STT_04 gồm có 2 thư mục con là: Noi dung và Ma Nguon.
File thư mục Noi dung có chức năng là: chứa bản báo cáo luận văn tốt nghiệp đề tài cảnh báo nhân viên ngủ gật dạng file word và file pdf.
File thư mục Ma nguon chứa 2 file đuôi *.py của ngôn ngữ python bao gồm: train.py và test_data.py.
Train là file xây dựng mô hình CNN và kết quả của mô hình.
Test_data là file real time và thực hiện đánh giá tình trạng của nhân viên, sau đó đưa ra cảnh báo nếu nhân viên ngủ gật sau thời gian 3 giây.
